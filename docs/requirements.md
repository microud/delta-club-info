# Delta Club Info - 需求文档

## 1. 项目概述

### 1.1 项目定位
一个专注《三角洲行动》（Delta Force: Hawk Ops）陪玩俱乐部的**中立信息站 + 社区平台**，类似"大众点评"在三角洲陪玩领域的垂直版本。

核心目标：帮助玩家快速找到靠谱俱乐部、查看真实评测、对比价格和服务，避免被坑。同时为俱乐部提供精准引流渠道。

### 1.2 目标用户
- **普通玩家**：浏览俱乐部、筛选玩法、查看评测视频、阅读用户评价、横向对比
- **俱乐部运营方**：通过付费推广获得曝光

### 1.3 业务背景

#### 三角洲行动烽火地带
烽火地带是《三角洲行动》中以"搜打撤"为核心的玩法模式（类似《逃离塔科夫》），玩家需在游戏中搜索物资、射击对手、成功撤离。

#### 哈夫币 (HafuCoin)
游戏内主要货币，可通过任务、战斗、搜刮获取。玩家间可以按一定比例将哈夫币兑换为人民币。

#### 俱乐部服务类型
- **跑刀**：不带装备进图，快速搜集物资后撤退。服务本质是"人民币 → 哈夫币"的兑换
- **陪玩**：高手陪打，按小时计费，技术水平分档（娱乐、初级技术、高级技术等）
- **护航体验单**：高手带队保护玩家撤离，价格较实惠的入门体验
- **护航标准单**：标准护航服务，"人民币 → 哈夫币"
- **护航趣味玩法**：有特定挑战的护航服务，可能附带保底机制和特殊规则

## 2. 核心功能

### 2.1 消费者端（微信小程序 + 公众号）

#### 用户认证
- 进入小程序即静默登录（`wx.login` → openid → 建用户记录），纯浏览不弹授权
- 触发收藏时弹出微信授权获取昵称和头像

#### 首页（推广 + 多 Tab Feed 流）
- 顶部轮播 Banner：所有生效推广订单对应的俱乐部海报，推广统一定价，轮播随机顺序，点击跳转俱乐部详情
- 多 Tab Feed 流：
  - 推荐 Tab：全类型内容混排，按发布时间倒序
  - 评测 Tab：仅评测 (REVIEW) 类内容
  - 舆情 Tab：仅舆情 (SENTIMENT) 类内容
  - 公告 Tab：俱乐部公众号公告 (ANNOUNCEMENT) + 系统公告
- 内容卡片：封面图 + 标题 + 博主名 + 平台标识（多平台聚合时显示多个平台图标）+ 关联俱乐部名 + 类型标签，点击进入内容详情页
- 公告卡片：标题 + 摘要 + 发布时间，点击进入公告详情页
- 首页核心作用是曝光换流量，不放俱乐部列表

#### 内容详情页
- 标题、简介、AI 摘要（LLM 未就绪前展示简介）
- 关联俱乐部跳转链接
- 多平台源列表（聚合内容展示所有平台入口，类似豆瓣影视多平台链接）
- 外链按钮（跳转原平台：B站/抖音/小红书/视频号/公众号）

#### 俱乐部列表页
- 卡片展示：Logo + 名称 + 运营时长 + 服务类型标签，不展示价格
- 排序：按创建时间倒序
- 筛选条件：服务类型（多选）
- 搜索：俱乐部名称模糊搜索，搜索与筛选可叠加
- 点击进入俱乐部详情页

#### 俱乐部详情页
- 顶部固定区域：Logo、名称、简介、运营时长、公众号/小程序导流、前身俱乐部链接、收藏按钮
- Tab 区域（左右滑动/点按切换）：
  1. 服务价格表（默认 Tab）：按服务类型分 Tab + 右上角"查看规则"弹 Modal（AI 高亮标注）+ 底部订单海报
  2. 评论：Stage 6 占位
  3. 评测内容：关联的 REVIEW 类内容列表（视频、笔记等多平台）
  4. 舆情内容：关联的 SENTIMENT 类内容列表（视频、笔记等多平台）
  5. 工商信息：公司全称、法人、经营状态等

#### 俱乐部对比页（推迟）
- 从列表页多选进入，不限数量，横向滚动
- 对比维度：价格、服务项

#### 俱乐部墓碑页（推迟）
- 展示所有已倒闭俱乐部
- 按倒闭时间排列
- 展示运营时长、倒闭备注、转生去向

#### 个人中心
- 顶部用户信息：微信头像 + 昵称（未授权时显示默认头像 + "点击登录"）
- 收藏的俱乐部（收藏列表页，展示方式同俱乐部列表）
- 我的评价（Stage 6 占位）

#### 评价系统
- 用户可发表评价（总分 + 服务/技术/性价比子维度）
- 其他人可点赞点踩、表达 Emoji
- 不支持互相回复（仅发表观点，非社交讨论）
- AI 自动审核，审核失败进人工队列

### 2.2 管理后台（Admin）

#### 俱乐部管理
- 俱乐部 CRUD（名称、Logo、简介、公众号、小程序等）
- 工商信息维护（公司全称、统一社会信用代码、法人、注册地址、经营范围、注册资本、公司成立日期、经营状态）
- 生命周期管理（发布、标记倒闭、关联前身俱乐部、填写成立日期）
- 服务项管理（跑刀/陪玩/护航体验/护航标准/护航趣味，各自价格结构，支持辅助图片）
- AI 智能录入：上传图片或粘贴文本 → AI 解析服务定价 → 用户确认后批量创建（护航趣味单解析为尽力而为，允许失败后手动录入）
- 规则管理（录入规则文本 → AI 分析利弊）
- 推广订单管理
- 订单海报管理（上传多张俱乐部海报图片，存储在 OSS）
- 公众号头像一键获取（根据公众号 ID 自动抓取头像上传）

#### 博主管理
- 博主列表 CRUD，展示关联的所有平台账号
- 新增/编辑博主时管理多个平台账号（B站、抖音、小红书、视频号）
- 每个账号配置：平台、平台用户 ID、采集分类（评测/舆情 多选）
- 启用/停用

#### 爬虫管理
- 采集任务列表，支持按平台/类型/状态筛选
- 单个任务可编辑 cron 表达式、启用/停用
- 执行记录列表
- 手动触发单个任务

#### 内容管理（原视频管理）
- 内容列表（筛选：平台、内容类型、分类、聚合状态、是否已关联俱乐部、AI 解析状态）
- 手动关联俱乐部
- 手动合并/拆分跨平台内容组
- 手动添加内容 URL

#### 企业微信集成与 AI 解析任务
- 通过企业微信自建应用接收运营者转发的俱乐部推广图片/消息
- 同一用户 30 秒内的消息自动分组为一个解析任务
- AI 自动解析图片和文本，提取俱乐部名称、服务定价、规则信息
- 解析任务列表管理（按状态筛选：pending / parsing / completed / failed）
- 解析任务审核页面：左侧原始图片，右侧 AI 结构化结果（可编辑修正）
- 确认后关联到已有俱乐部或新建俱乐部，写入服务项和规则

#### 评论管理
- 评论列表（筛选：状态、俱乐部）
- AI 审核失败的人工审核队列
- 批量通过/拒绝

#### 系统设置
- 管理员账号管理
- AI Provider 配置（支持 xAI/OpenAI/Anthropic 多供应商，API Key + 模型选择）
- 企业微信配置（企业 ID、应用 Secret、回调 Token、EncodingAESKey）
- TikHub API 配置（API Key、速率限制）
- Emoji 表情管理

### 2.3 数据自动化

#### 多平台内容采集（三条任务线，通过 TikHub 平台统一接入）
- **评测内容 (REVIEW)**：按指定博主列表抓取最新内容（B站、抖音、小红书、视频号）
- **舆情内容 (SENTIMENT)**：按俱乐部名称关键词搜索 + 指定舆情博主采集（B站、抖音、小红书、视频号）
- **俱乐部公告 (ANNOUNCEMENT)**：按俱乐部官方公众号拉取最新文章（仅微信公众号）
- 博主支持多平台账号管理，每个账号可标记参与评测/舆情采集
- 每个采集任务独立 cron 表达式，默认每小时，Admin 可单独调整
- 跨平台内容聚合：AI 识别同一内容在不同平台的副本，以统一入口展示多平台源

#### AI 解析
- 输入：视频标题 + 简介（字幕抓取暂不实现）
- 输出：匹配俱乐部名、简要评价、情感倾向
- 支持多 AI Provider（xAI Grok / OpenAI / Anthropic），通过 AI SDK 统一接入
- 使用 Vercel AI SDK (`ai` + `@ai-sdk/xai` / `@ai-sdk/openai` / `@ai-sdk/anthropic`)

#### 规则 AI 分析
- 输入：俱乐部规则原文
- 输出：对玩家有利/不利/中性标注
- 高亮展示帮助用户快速识别规则利弊

#### 服务定价 AI 解析（智能录入 + 企业微信）
- 输入：俱乐部推广图片 + 可选文本
- 输出：俱乐部名称、服务列表（类型/价格/保底/规则）、通用规则
- 使用 Grok Vision 多模态能力处理图片
- 结构化输出（AI SDK structured output + Zod schema）

### 2.4 商业化
- 付费推广：俱乐部通过线下联系付费，Admin 手动录入推广订单
- 推广排序：按所有生效订单的日均费用 (dailyRate) 之和排序
- 支持多次加价（叠加新订单），不需要取消旧订单
- 不做俱乐部自主认领和自助购买
