# Delta Club Info - 需求文档

## 1. 项目概述

### 1.1 项目定位
一个专注《三角洲行动》（Delta Force: Hawk Ops）陪玩俱乐部的**中立信息站 + 社区平台**，类似"大众点评"在三角洲陪玩领域的垂直版本。

核心目标：帮助玩家快速找到靠谱俱乐部、查看真实评测、对比价格和服务，避免被坑。同时为俱乐部提供精准引流渠道。

### 1.2 目标用户
- **普通玩家**：浏览俱乐部、筛选玩法、查看评测视频、阅读用户评价、横向对比
- **俱乐部运营方**：通过付费推广获得曝光

### 1.3 业务背景

#### 三角洲行动烽火地带
烽火地带是《三角洲行动》中以"搜打撤"为核心的玩法模式（类似《逃离塔科夫》），玩家需在游戏中搜索物资、射击对手、成功撤离。

#### 哈夫币 (HafuCoin)
游戏内主要货币，可通过任务、战斗、搜刮获取。玩家间可以按一定比例将哈夫币兑换为人民币。

#### 俱乐部服务类型
- **跑刀**：不带装备进图，快速搜集物资后撤退。服务本质是"人民币 → 哈夫币"的兑换
- **陪玩**：高手陪打，按小时计费，技术水平分档（娱乐、初级技术、高级技术等）
- **护航体验单**：高手带队保护玩家撤离，价格较实惠的入门体验
- **护航标准单**：标准护航服务，"人民币 → 哈夫币"
- **护航趣味玩法**：有特定挑战的护航服务，可能附带保底机制和特殊规则

## 2. 核心功能

### 2.1 消费者端（微信小程序 + 公众号）

#### 首页（推广 + 运营位）
- 推广俱乐部轮播/卡片（付费曝光，明牌广告）
- 精选评测视频推荐
- 公告/活动
- 首页核心作用是曝光换流量，不放俱乐部列表

#### 俱乐部列表页
- 筛选条件：服务类型、价格排序（汇率性价比、小时单价）
- 搜索：俱乐部名称模糊搜索
- 多选模式：选中多个俱乐部后进入对比页
- 点击进入俱乐部详情页

#### 俱乐部详情页
- 基本信息（Logo、简介、运营时长、公众号/小程序跳转导流到俱乐部自己的渠道）
- 前身俱乐部链接（秽土转生历史链）
- 服务价格表（按类型分 Tab 展示）
- 规则展示（AI 标注高亮：绿色=有利、红色=不利、灰色=中性）
- 评测视频列表（跳转 B 站/抖音播放）
- 舆情视频列表（单独区域，标注情感倾向）
- 用户评价区（评分统计 + 评价列表 + Emoji 互动）

#### 俱乐部对比页
- 从列表页多选进入，不限数量
- 对比维度：价格、服务项

#### 俱乐部墓碑页
- 展示所有已倒闭俱乐部
- 按倒闭时间排列
- 展示运营时长、倒闭备注、转生去向

#### 个人中心
- 收藏的俱乐部
- 我的评价
- 微信头像/昵称

#### 评价系统
- 用户可发表评价（总分 + 服务/技术/性价比子维度）
- 其他人可点赞点踩、表达 Emoji
- 不支持互相回复（仅发表观点，非社交讨论）
- AI 自动审核，审核失败进人工队列

### 2.2 管理后台（Admin）

#### 俱乐部管理
- 俱乐部 CRUD（名称、Logo、简介、公众号、小程序等）
- 生命周期管理（发布、标记倒闭、关联前身俱乐部、填写成立日期）
- 服务项管理（跑刀/陪玩/护航体验/护航标准/护航趣味，各自价格结构）
- 规则管理（录入规则文本 → AI 分析利弊）
- 推广订单管理

#### 博主管理
- 博主列表 CRUD
- 启用/停用

#### 爬虫管理
- 爬虫任务执行记录
- 抓取频率配置（默认每小时）
- 手动触发抓取

#### 视频管理
- 视频列表（筛选：平台、分类、是否已关联俱乐部、AI 解析状态）
- 手动关联俱乐部
- 手动添加视频 URL

#### 评论管理
- 评论列表（筛选：状态、俱乐部）
- AI 审核失败的人工审核队列
- 批量通过/拒绝

#### 系统设置
- 管理员账号管理
- LLM Provider 配置
- 爬虫频率全局设置
- Emoji 表情管理

### 2.3 数据自动化

#### 视频采集（两条任务线）
- **评测/体验视频 (REVIEW)**：按指定博主列表抓取最新视频
- **舆情视频 (SENTIMENT)**：按俱乐部名称关键词搜索最新视频
- 平台：B 站 + 抖音
- 频率：默认每小时，Admin 可调

#### AI 解析
- 输入：视频标题 + 简介 + 字幕文本
- 输出：匹配俱乐部名、简要评价、情感倾向
- 使用 xAI Grok，抽象为 LLM Provider 可替换

#### 规则 AI 分析
- 输入：俱乐部规则原文
- 输出：对玩家有利/不利/中性标注
- 高亮展示帮助用户快速识别规则利弊

### 2.4 商业化
- 付费推广：俱乐部通过线下联系付费，Admin 手动录入推广订单
- 推广排序：按所有生效订单的日均费用 (dailyRate) 之和排序
- 支持多次加价（叠加新订单），不需要取消旧订单
- 不做俱乐部自主认领和自助购买
